# Dream In Mocha Mauve

A YASB theme based on the **Catppuccin Mocha** palette with **Mauve** accent (`#cba6f7`). It features translucent backgrounds, subtle borders, and smooth animations.

## Preview

![Dream In Mauve](YASB%20Dream%20In%20Mocha%20Mauve.png)

## Widgets

- **Left** — Home, Clock, Volume, GlazeWM Workspaces, Active Window.
- **Center** — Media.
- **Right** — Grouper (CPU, Memory, GPU), GlazeWM Tiling Direction, GlazeWM Binding Mode, Quick Launch, Wallpapers, Notifications, Power Menu.

## Features

- Based on the **Catppuccin Mocha** palette with semi-transparent colors.
- The **Mauve** accent unifies the entire interface.
- Translucent backgrounds with blur effects are applied across the bar and all popups.
- Smooth transitions on hover states.

## Configuration Tips

- **Home Widget** — If you don't use OneDrive, remove `OneDrive\` from the paths in `config.yaml` (e.g. `%USERPROFILE%\Desktop` instead of `%USERPROFILE%\OneDrive\Desktop`)
- **Wallpapers Widget** — Change `image_path` in `config.yaml` to point to your own wallpapers directory (default: `C:\Wallpapers`)

## Dependencies

- **YASB**
- **GlazeWM**
- **JetBrains Mono Nerd Font**
